# App Layout

Use this folder to define the app structure before building UI.

## Files

- `app-structure.md`: navigation, windows, and major sections
- `screen-inventory.md`: per-screen purpose and contents
- `wireframe-spec.md`: sizing, spacing, and pane behavior rules
- `references/main-screen.png`: visual reference for the primary desktop shell

## Notes

- Treat this as the source of truth for UI shape.
- Treat `references/` as supporting visual guidance, not the source of truth for behavior or product decisions.
- Update it when flows or screen responsibilities change.
